HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web . Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da ingestão entre os padrões HyTime e SGML. Hytime é um padrão para a representação estruturada de hipermídia e conteúdo baseado em tempo. Os Hipertexto são a entusiasmados de vários elementos - palavras, vídeos e conteúdos-, que, qundo conectados, estabelencem uma rede de dados que permite a comunicação, o armazenamento e o compartilhamento de informações. Por exemplo, ao visitar um site, você encontra diversas informações com informações diferentes, como parágrafos, marcadores e fontes distintas. Pois então, essa estrutura é realizada por meio do HTML.

Qual a Função do HTML?

Imagine que você precisa construir um site do zero com informações da sua empresa. nesse caso, você vai conhecer a estrutura do HTML, criando um documento no bloco de notas e acrescentando codigos-tags-, que indicam paragrafos, quebra de texto espaçamento, imagens, entre outros. Em geral, em um único site, existem diversos documentos HTML, criando páginas como homepage, página de produto e informações sobre a empresa. Em cada aba, existe um código diferente.

    Os 5 principais protocolos

  1. HTTP - Protocolo de transferência de hipertexto
  2. FTP - Protocolo de transferência de arquivos
  3. POP - Post Office Protocol
  4. SMTP - Protocolo Simples de Transferência de Correio
  5. IMAP - Internet Messaging Access Protocol


O que é CSS?

Cascading Style Sheets (CSS) é usado para formatar ou layout de uma página da web. Com CSS, você pode controlar a cor, a fonte, o tamanho do texto, o espaçamento entre os elementos, como os elementos são posicionais e de mesa, quais imagens de fundo ou cores de fundo devem ser usadas, exibições diferentes para dispositivos e tamanhos de tela diferentes e muito mais!

O que é HTML?

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da ingestão entre os padrões HyTime e SGML. HyTime é um padrão para uma representação estruturada de hipermídia e conteúdo baseado em tempo.

O que é JavaScript?

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.
